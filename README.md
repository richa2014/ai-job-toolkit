# AI Job Toolkit

An open-source collection of AI tools, prompts, templates, and workflows for developers and job seekers.

## What is inside?

- Resume scoring prompts
- Job description analyzer
- AI workflow examples
- Documentation templates
- Developer productivity tools

## Why this project?

Building AI projects is hard. This repository aims to provide reusable templates and examples that help developers build faster.
