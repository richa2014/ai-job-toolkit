# AI Interview Question Generator

Generate interview questions from a job description.

## Features

- Generate technical questions.
- Generate behavioral questions.
- Support beginner, intermediate, and advanced levels.
- Export questions as Markdown.

## Example Prompt

Input:

```text
Create interview questions for a React developer with 3 years of experience.
```

Output:

- Explain React hooks.
- What is state management?
- Describe a difficult bug you fixed.
- How do you optimize performance?

## Goal

Help candidates prepare for interviews using AI.
