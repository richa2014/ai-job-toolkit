# Cover Letter Generator Prompt

## Goal

Generate a professional and personalized cover letter using a resume and job description.

## Inputs

- Candidate name
- Resume details
- Skills
- Experience
- Job title
- Company name
- Job description

## Instructions

Create a cover letter that:

- Matches the candidate's experience to the job.
- Highlights relevant skills.
- Uses a professional tone.
- Stays concise and easy to read.
- Avoids generic statements.

## Example Prompt

Using the resume and job description below, generate a personalized cover letter.

<Resume>

<Job Description>

## Output Format

Return the result in Markdown format.

## Notes

The cover letter should be clear, reusable, and customizable.
